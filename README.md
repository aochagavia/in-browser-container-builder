# In-browser container builder

> Build container images and download them as a tar archive

This is demo-quality software, meant as a companion to [this blog post](TODO).
The code is a bit messy, but the UI is straightforward. The goal is to showcase the possibilities of custom container tooling and to spark your curiosity, not to explain low-level container fundamentals.

_Important: this is a research prototype, don't use it for anything serious. If you need something production-ready, let's talk._

### Development

Run locally using `npm run dev`
